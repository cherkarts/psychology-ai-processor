
=== ИНСТРУКЦИЯ ПО НАСТРОЙКЕ КОММЕНТАРИЕВ ===

1. ПОДКЛЮЧЕНИЕ К БАЗЕ ДАННЫХ:
   - Убедитесь, что MySQL запущен
   - Проверьте настройки в config.php
   - Примените миграции: php database/apply-comments-migration.php

2. НАСТРОЙКА TELEGRAM БОТА:
   - Создайте бота через @BotFather
   - Получите токен бота
   - Настройте webhook для уведомлений

3. ИНТЕГРАЦИЯ НА СТРАНИЦАХ:
   - Добавьте виджет комментариев на страницы статей
   - Добавьте виджет авторизации Telegram
   - Настройте модерацию в админке

4. ПРОВЕРКА РАБОТЫ:
   - Откройте страницу статьи
   - Проверьте авторизацию через Telegram
   - Проверьте добавление комментариев
   - Проверьте модерацию в админке

5. НОЧНАЯ ТЕМА:
   - Проверьте работу на страницах статей
   - Убедитесь, что стили применяются корректно
   - Проверьте переключение темы

=== ФАЙЛЫ ДЛЯ ПРОВЕРКИ ===
- admin/comments.php - модерация комментариев
- api/comments.php - API для комментариев
- includes/comments-widget.php - виджет комментариев
- includes/telegram-auth-widget.php - авторизация Telegram
- database/comments.sql - структура базы данных
